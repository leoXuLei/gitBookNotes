创建仓库

本地项目与GitHub关联

项目内执行命令，如下:

git init // 为书籍项目创建本地仓库
git add .
git commit -m "init project"
git remote add origin https://github.com/xxxxx.git // 与远程仓库关联
git push -u origin master
